# Components

## Common Components
Các UI components tái sử dụng trong toàn dự án.

### Đã có:
- `Button` - Component button với nhiều variants
- `Input` - Component input với label và error handling

### Cần thêm:
- Modal
- Card
- Table
- Pagination
- Loading Spinner
- Alert/Toast
- Select/Dropdown
- DatePicker
- Form

## Feature Components
Các components đặc thù cho từng feature.

### Ví dụ:
- `RoomCard` - Hiển thị thông tin phòng trọ
- `TenantList` - Danh sách khách thuê
- `PaymentTable` - Bảng thanh toán
- `ContractForm` - Form tạo/sửa hợp đồng
